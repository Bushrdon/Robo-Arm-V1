# Notes

La estructura mecanica aun no es funcional en lo mas minimo. Se puede considerar una prueba de concepto por los momentos. 
La idea principal con esta estructura es integras las articulaciones a las partes extensibles del brazo de modo que las piezas no sean muchas y se tenga un diseno simple pero eficiente en cuanto a extensibilidad y movimiento. Las piezas fueron tomadas temporalmente de este [repositorio](https://github.com/JasonHEngineering/DIY-Robot-Arm-V2/tree/main/CAD%20files%20(2d%2C%20step)), proximamente se le haran las modificaciones necesarias para que sirva al modelo de los actuadoresy al modelo del robot en general.  
Ningun modelo CAD de ninguna pieza oficial del robot se ha compartido hasta el momento.
